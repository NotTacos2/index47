NextFTC
NextFTC is a collection of command based libraries developed by Rowan McAlpin, Davis Luxenberg, and Zach Harel. NextBindings provides complex gamepad integration, and NextControl has PID/PIDF control, feedforward, motion profiling, and Kalman filters. NextFTC is written in Kotlin with Java support, and has extensions to work with PedroPathing and RoadRunner. More information and documentation can be viewed at nextftc.dev.
