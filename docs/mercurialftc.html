MercurialFTC
MercurialFTC Mercurial is a Command Based library written by Oscar Chevalier in Kotlin with Java support. It has a large focus on ease of use in comparison to FTCLib, with nearly no need to write Commands as Classes, helpful error messages for exceptions in Commands, etc. It is part of the Dairy ecosystem of open source FTC libraries, including Sinister (classpath scanning utility library), Core (runtime extensions), and Pasteurized (gamepad management library). More information and Documentation can be viewed at docs.dairy.foundation/Mercurial/overview.
